# StudyPulse - Quick Start Guide

Get started tracking your study performance in 5 minutes!

## Installation (2 minutes)

```bash
# Clone or extract the project
cd studypulse

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:3000`

## Your First Study Session (1 minute)

1. Click the **"Track"** tab
2. In the form on the left:
   - **Subject**: Type "Math" (creates new subject)
   - **Duration**: 45 (minutes)
   - **Date**: Today (pre-filled)
   - **Notes**: Optional - "Chapter 5: Calculus"
3. Click **"Log Session"**

âœ… **Done!** Your first session is logged.

## Check Your Progress (1 minute)

1. Look at the top of the page - see your **Total Study Hours** updated!
2. Click **"Overview"** tab to see charts
3. Your session appears in the "Recent Sessions" list

## Set Your First Goal (1 minute)

1. Click the **"Goals"** tab
2. In the form:
   - **Subject**: Math
   - **Weekly Target**: 5 (hours)
   - **Study Days**: Click Mon-Fri (5 days)
3. Click **"Create Goal"**

âœ… **Bonus!** You can now see your progress toward this goal.

## Key Navigation

| Tab | Purpose |
|-----|---------|
| **Overview** | View analytics and trends |
| **Track** | Log new study sessions |
| **Goals** | Create and monitor goals |
| **Analytics** | Detailed performance charts |
